Willkommen

Diese Applikation wurde im Rahmen des CAS in Frontend Engineering 2020 an der HSR erstellt. Die Aufgabe bestand darin, eine Notizapplikation mit den Komponenten JavaScript, CSS und HTML zu erstellen. Dabei wurden keine Frameworks oder Styling Bibliotheken verwendet.

Funktionen

Der Umfang der Applikation wurde relativ gering gehalten und orientiert sich an einer «Mini-Todo-App». Es können neue Notizen hinzugefügt werden, welche dann auf der Übersichtsseite dargestellt werden. Diese können von dort aus abgeschlossen, editiert und gelöscht werden. Die abgeschlossenen Notizen können ein- & ausgeblendet werden. Ebenfalls kann auf der Übersichtseite eine gewünschte Sortierung ausgewählt werden.
